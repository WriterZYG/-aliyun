舆情大屏产品说明及操作指导

一、 产品介绍

以数品牌舆情大屏是一款可视化数据动态展示软件，可投放于PC端、TV端、大屏端；已接入阿里云公众趋势分析API数据接口，用户可直接通过阿里云账号登录进行品牌舆情相关操作；可实时监控当日品牌舆情热点，营销事件追踪分析、竞品舆情分析等品牌全域核心数据。

以数品牌舆情大屏是基于大数据的可视化平台，实现了品牌对全网舆情数据的实时监控，全域营销动态全盘掌握及竞品动态一览。

二、 产品功能概述

品牌舆情产品是用户观察互联网以及感知品牌营销及舆情状况的入口，产品涵盖三条主线：覆盖主流新闻媒体网站进行舆情采集、专题及关键词的多维度分析、媒体传播情感趋势以及关键节点分析。

产品除了前端大屏展示外共分为4大功能，分别是主控制台、舆情盒子、品牌管理以及传播分析：

1. 主控制台

主控制台是后台展示功能的驾驶舱，同时也是三块大屏的入口，通过主控制台，用户可以看到品牌专题、关键词以及微博分析的总数。

2. 舆情盒子

舆情盒子可以实时展示舆情信息，根据设置的关键词进行舆情的采集、分析，最后将结果汇总展示。一般情况下，在设置完关键词后的5到10分钟后即可看到舆情盒子中的内容，每条舆情信息都可以点击并跳转至源链接地址以供查看。

3. 品牌管理

品牌管理分为自身品牌设置、关注品牌设置以及品牌位置设置三个功能。通过对自身品牌以及关注品牌的名称、关键词以及位置等信息的设置，即可完成品牌管理的核心设置。设置完成后，主控制台、舆情盒子以及大屏会在一定的时间内关联及展示相关的舆情数据。

4. 传播分析

传播分析为用户提供了对微博进行深度分析的能力。通过对微博账号地址的分析，识别微博传播的核心节点以及传播层级，区分用户属性、账号类别以及地域等信息。根据微博转发量的不同，微博传播的分析时间也会不同，从几分钟到几十分钟不等。

2. 品牌管理

首次进入“品牌舆情大屏”管理控制台，需要为分析对象配置品牌、关键词以及位置等信息。分析对象可以是企业或品牌，可以用一个或多个关键词组合进行描述。如用户关注的品牌是“互联网金融”，则可能会定义的关键词有“陆金所”、“支付宝 招财宝”、“宜人贷”等等。除了自身品牌，系统还支持配置7个其他关注品牌。

系统后台将采集包含这些品牌名称和关键词组合的文章内容，进行汇总分析。后续一些统计图表也将主要针对监控品牌及关键词进行分析。

5. 可视化大屏

完成品牌管理以及传播分析的设置以后，在主控制台点击页面下方的“自身品牌舆情”、“实时要闻传播分析”以及“关注品牌分析”三块大屏的缩略图，即可切换并跳转至可视化大屏页面，在PC端分别展示这三块大屏，查看分析结果。

特别提醒：由于后台需要采集、分析以及汇总计算，所以可视化大屏的数据并不会立刻更新，需要等待1小时左右。
